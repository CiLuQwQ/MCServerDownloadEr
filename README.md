<h1 align="center">MC服务端下载器</h1>
<p align="center">
这是一个命令行Minecraft服务端下载器，可下载各种服务端

</p>

### 运行
需要Java17才可正常运行。使用java -jar 文件名 运行

### 下载API
原版服务端国内源由BMCLAPI和MCBBS提供高速下载服务.BMCLAPI为公益服务.赞助BMCLAPI可以帮助作者提供更好的高速下载服务
赞助BMCLAPI:https://afdian.net/a/bangbang93
其他请见docs目录下的API.txt
